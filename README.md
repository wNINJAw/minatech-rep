## 📖 Sobre o projeto

O **MinaTech** é um site institucional desenvolvido para a disciplina de Desenvolvimento Web.

O objetivo do projeto é apresentar a organização, sua missão, pilares, parceiros e formas de participação por meio de uma interface moderna, responsiva e intuitiva.

---

- Landing Page moderna
- Sessão Hero
- Sobre a organização
